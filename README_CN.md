## 💻 关于本项目 (About This Resume Project)

这是一个基于现代前端技术栈构建的在线交互式简历（Candidate Brief）平台。

### ✨ 特性 (Features)
*   🌍 **多语言支持 (i18n)**: 一键切换中文 (ZH)、英文 (EN)、日文 (JA)。
*   📱 **响应式设计**: 完美适配桌面端和移动端浏览。
*   🖨️ **打印友好**: 深度优化了 `@media print` 样式，支持一键导出排版精美的 PDF。
*   ⚡ **现代技术栈**: React 19 + Vite + Tailwind CSS v4 + Framer Motion。

### 🛠️ 本地运行 (Local Development)

```bash
# 1. 安装依赖
npm install

# 2. 启动开发服务器
npm run dev

# 3. 构建生产版本
npm run build
```
